@Documented指明需要在Javadoc中包含Annotation（缺省是不包含的）。使用@Documented的一个技巧就是指定Retention为RetentionPolicy.RUNTIME。这样，Annotation就会保留在编译后的类文件中并且由虚拟机加载，然后Javadoc就可以抽取出Annotation并添加到类的HTML文档中。
