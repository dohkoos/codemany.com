Ubuntu 8.10英文Locale下安装Fcitx中文输入法

Nov 17th, 2008

系统装好后首先要做的就是安装中文输入法。在网上找了一些中文输入法的资料，最常用的是Fcitx和SCIM输入法。不过好像SCIM的问题比较多，而且据说Fcitx的功能上也要比SCIM强得多，所以决定采用Fcitx输入法。

卸载SCIM
