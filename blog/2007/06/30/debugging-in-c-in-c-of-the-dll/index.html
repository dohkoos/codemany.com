在C#中调试C++的DLL

Jun 30th, 2007

在C#中调试C++的DLL很简单，只要在C#项目属性的Debug一项中钩上”Enable unmanaged code debugging”就OK了，然后你就可以像调试一般程序那样调试DLL了。
