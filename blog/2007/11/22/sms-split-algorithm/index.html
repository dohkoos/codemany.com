短信分拆算法

Nov 22nd, 2007

今天，突然心血来潮，写了一段分拆短信的代码。代码如下：

“` public class SMSHelper {

private static final int SMS_MAX_LENGTH = 70;

public static List split(String msg) {<br />
    if (msg.length() <= SMS_MAX_LENGTH) {<br />
        throw new IllegalArgumentException("被分拆的短信长度必须大于70个字符");<br />
    }

    List<String> list = new ArrayList<String>();<br />
    int i = 0, start = 0;<br />
    while (true) {<br />
        i++;<br />
        String prompt = "(第" + i + "条)";<br />
        int remain = SMS_MAX_LENGTH - prompt.length();<br />
        if (start + remain > msg.length()) {<br />
            list.add(prompt + msg.substring(start));<br />
            break;<br />
        }<br />
        list.add(prompt + msg.substring(start, start + remain));<br />
        start += remain;<br />
    }<br />
    return list;<br />
}<br />

} [/code]
