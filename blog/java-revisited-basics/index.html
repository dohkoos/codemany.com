Java Revisited：语言基础

Jan 10th, 2016 2:14 pm

注释

单行注释//；多行注释/* */，但不支持嵌套；文档注释/** */。

数据类型

Java是强类型语言，因此每个变量都必须声明类型。在Java中共有8种基本类型：4种整数类型，2种浮点数类型，字符类型和布尔类型各1种。

类型 所占字节 示例
byte 1
short 2
int 4 0xcafe, 010, 0b1001
long 8 4000000000L
float 4 3.14F
double 8
char 2 'a'

所有的浮点数计算都遵循IEEE 754规范。有三个特殊的浮点值（正无穷大、负无穷大、非数字）被用来表示溢出和错误：

如果基本的整型和浮点型数据无法达到要求的精度，那么可以使用java.math包中的BigInteger和BigDecimal两个大数字类。这两个类可以操作任意长的数字。BigInteger类实现了任意精度的整数运算，而BigDecimal实现了任意精度的浮点数运算。

运算符

  • 算术运算符+、-、*、/、%；
  • 递增/减运算符++、--；
  • 关系运算符==、!=、<、<=、>、>=；
  • 逻辑运算符&&、||、!，并且&&和||按短路方式计算；
  • 三元运算符?:；
  • 位运算符&、|、^、~、<<、>>、>>>逻辑右移，左边位填0；

下表是运算符的优先级：

运算符 结合性 备注
[] . () 从左向右 method call
! ~ ++ -- + - () new 从右向左 unary、postfix、cast
* / % 从左向右 multiplicative
+ - 从左向右 additive
<< >> >>> 从左向右 shift
< <= > >= instanceof 从左向右 relational
== != 从左向右 equality
& 从左向右 bitwise AND
^ 从左向右 bitwise exclusive OR
| 从左向右 bitwise inclusive OR
&& 从左向右 logical AND
|| 从左向右 logical OR
?: 从左向右 ternary
= += -= *= /= %= &= |= ^= <<= >>= >>>= 从右向左 assignment

数字类型间的转换

图中的6个实线箭头表示无信息损失的转换，而3个虚线箭头表示的转换可能丢失精度。在使用二元运算符对两个值进行计算之前，两个运算元都会转换成通用类型。以下的转换时的规则：

  • 如果两个运算元中有一个是double类型，则另一个将会转换成double类型；
  • 否则，如果两个运算元中有一个是float类型，则另一个将会转换成float类型；
  • 否则，如果两个运算元中有一个是long类型，则另一个将会转换成long类型；
  • 否则，两个运算元都将转换成int类型。

控制流程

复合语句是用一对花括号扩起来的任意数量的Java语句，它定义了变量的作用范围，可以嵌套。

在处理多种选择时，使用if/else结构非常麻烦，因此Java提供了专门用于处理多分支结构的switch语句，其中case标签只支持char、byte、short、int及其封装类，String类型和枚举类型：
