在Visual C#中调试C++的DLL

Jul 3rd, 2007

在Visual C#中调试C++的DLL很简单，只要在C#项目属性的Debug一项中钩上“Enable unmanaged code debugging”，然后你就可以像调试一般程序那样调试DLL了。
