如何实现点击链接下载文件时避免浏览器自动打开文件

Jun 3rd, 2012

HTTP Response Header有个Content-Disposition参数，该参数的目的是为了在客户端另存文件时提供一个建议的文件名。将它的
值设置为attachment后，浏览器就会老老实实地显示另存为对话框，如果这个值设成inline，则无论怎样浏览器都会自动尝试用已知关联的程序打开文件。

注意：在Firefox中需要把filename用双引号包起来才能得到想要的文件名，不然如果含有空格就会丢掉空格后面的部分，而IE会把空格转为下划线。

Ruby设置header代码
