PowerBuilder生成可执行文件的小技巧

Mar 29th, 2013

1、使用资源文件（.pbr）引用资源

在资源相同目录下建立.pbr文件，其中列出在应用中动态引用的资源文件，一行列出一个资源，格式如：

在.pbr文件中指定的文件名必须与在脚本中引用的资源匹配，若引用时包含路径，则在.pbr文件中也必须包含同一路径，否则因PowerBuilder在执行时只是简单地进行字符串比较而导致无法发现该资源。

2、设置可执行文件图标

应用 -> 属性 -> General -> Additional Properties按钮 -> Icon标签页 -> Icon Name

3、可执行文件所需的环境DLL

最简便的方式是使用自带的打包程序PowerBuilder Runtime Packager。这个应用会自动把脱离环境运行所需的DLL等文件筛选出来。运行后会生成一个MSI应用，执行下就可以解压出需要的DLL文件了。
